# 📦 CodeStock

Sistema web para identificação e controle de matéria-prima, com foco na padronização de etiquetas e organização de estoque.

---

## 📄 Sobre o projeto

O **CodeStock** é um sistema desenvolvido para melhorar o processo de identificação, cadastro e controle de matérias-primas em ambientes industriais.

O projeto surgiu a partir de um problema real observado na empresa **Panasonic do Brasil**, onde falhas no processo de identificação e registro de materiais geram erros operacionais, retrabalho e baixa confiabilidade das informações.

Atualmente, o sistema está em fase inicial, com interface web construída em HTML, simulando as principais funcionalidades do controle de estoque.

---

## ⚠️ Problema

O processo atual de controle de matéria-prima apresenta diversas falhas:

* Etiquetas incompletas ou ilegíveis
* Registros feitos manualmente
* Troca de códigos de materiais
* Duplicidade de estoque
* Demora na conferência
* Falta de padronização

Esses problemas impactam diretamente a organização do estoque e a confiabilidade dos dados.

---

## 💡 Solução proposta

O CodeStock propõe a criação de um sistema web para:

* Padronizar o cadastro de matérias-primas
* Melhorar a identificação dos materiais
* Organizar as informações de estoque
* Reduzir erros operacionais

Como evolução do projeto, está prevista a implementação de:

* Geração de etiquetas automáticas
* Uso de QR Code para identificação
* Consulta rápida de materiais
* Registro de movimentações

---

## 🎯 Objetivo

Desenvolver um sistema web que melhore a organização, identificação e controle de matérias-primas, reduzindo erros e aumentando a eficiência dos processos.

---

## 🛠️ Funcionalidades atuais
